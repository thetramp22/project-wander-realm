# Glossary

|Term                   |Meaning                |
|:---                   |:---                   |
|World                  |The authoritative state of the simulation.|
|Simulation |The system responsible for advancing time and updating the world.|
|Entity |An object that independently exists within the running simulation.|
|Definition |Static data describing a type of object.|
|Instance |A runtime object existing within the simulation.|
|Container |An object capable of holding item instances.|
